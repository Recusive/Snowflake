## Installation With Package Manager

Snowflake is available in below software repositories:

[![Packaging status](https://repology.org/badge/vertical-allrepos/snowflake.svg)](https://repology.org/project/snowflake/versions)

Snowflake is also additionally available via:

- [Flatpak](https://flathub.org/apps/details/dev.snowflake.snowflake)
- [Scoop](https://scoop.sh/#/apps?q=snowflake)


For Fedora Linux, Snowflake is (unofficially) available via dnf RPM package manager, using nightly as of 2024.01:

- [copr](https://copr.fedorainfracloud.org/coprs/titaniumtown/snowflake/)

```bash
sudo dnf copr enable titaniumtown/snowflake 
sudo dnf install snowflake
```
